Project Description

Welcome to our Senior/Graduate Project page. The focus of this project is to utilize a large set of data and learn to visualize it using one of the many visualization languages. For the sake of this project, we have chosen to use D3 and javascript. The data we have chosen to visualize contains information on the universiies across the U.S. We are going to create some form of an interactive map that users to access the data visually. We will update this page when we begin to implement our prototype.
